Decentralized. Storage.

Syndicate integrates cloud services into a scalable coherent storage system. This puts you in control of your data, not the services.




Features

Decentralized

With Syndicate, you are not tied down to any single service provider, and you can seamlessly move between them. You control where and how your data is stored.

Secure

Syndicate offers a secure programmable storage layer over untrusted providers. You control what Syndicate does with your data, and who can access it.

Fast

Syndicate can use content distribution networks and scalable PaaS providers to boost the speed of your data downloads, while also ensuring you always read fresh, correct data.

Developer API

Syndicate gives developers end-to-end control over data consistency and trust boundaries, so you never worry about hosting user data or scaling access to it.

Research

Syndicate is scalable software-defined storage. With Syndicate, you control end-to-end data consistency, dataflow processing, and admin/trust boundaries across a changing set of existing cloud services. With Syndicate, each user gets their own decentralized storage volumes that can host a scalable amount of data and handle a scalable number of reads and writes. Volumes are always online and are not tied to any service providers--users do not need to run or manage their own servers, nor do they need to leave any of their devices turned on 24/7. Syndicate instead incorporates existing cloud storage, content distribution networks, publicly-routable datasets, PaaS providers, and authentication services to create volumes.

Please read our extended abstract or whitepaper for a technical overview of the Syndicate architecture.


Applications that are being built using Syndicate include:


Secure Email

Storing emails in the cloud is convenient and enables access from multiple devices. However, users have to trust cloud providers for not leaking or processing their emails. Syndicate enables private encrypted email that is secure as PGP email, but without forcing users to manage keys.

Scalable Scientific Data Processing

Scientific data storage is the ultimate data silo. Syndicate helps integrate scientific storage with commodity CDNs and cloud storage to enable multiple labs to efficiently share data between their computing facilities.

End-to-End Encrypted Shared Folders

Dropbox showed the world the value of having a shared folder sync'ed across all your devices. With Syndicate, your shared folder can not only scale to millions of users, but also offer end-to-end encryption and arbitrary replication policies.



About

Syndicate is developed at the Computer Science department of Princeton University. Syndicate started off as an academic research project by Jude Nelson under the supervision of Larry Peterson. Syndicate is currently deployed using PlanetLab and VICCI and contributors include: Muneeb Ali, Wathsala Vithanage, John Whelchel, and Illyoung Choi.
